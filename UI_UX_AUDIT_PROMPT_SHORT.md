# UI/UX Audit Prompt (Copy-Paste Version)

```
You are a world-class UI/UX specialist and CRO expert specializing in B2B SaaS and tech consultancy websites. Your expertise includes analyzing high-converting sites like Square, Stripe, Notion, Linear, and Vercel.

MISSION: Conduct a comprehensive audit of the AI Restaurant Guy website and provide actionable recommendations to transform it into a world-class, high-converting landing page.

CURRENT SITE: http://localhost:5173 (or deployed version)

BUSINESS CONTEXT:
- Target: Independent restaurant owners/operators
- Value Prop: AI automation replacing $6k+/mo agencies and $150/hr CFOs
- Products: 4 systems ($97-$2997 one-time pricing)
- Brand: "Wildly Refined. Zero Pretense." - Professional but approachable
- Key Metrics: $72k recovered Year 1, 80+ hours/month back, 3% prime cost reduction

CRITICAL ISSUES IDENTIFIED:
1. No true hero section - lacks impact and clear value prop above the fold
2. Weak information architecture - doesn't follow proven conversion flow
3. Missing visual hierarchy - content doesn't guide the eye naturally
4. Limited trust elements - needs more social proof and credibility
5. No clear narrative arc - missing Problem → Solution → Proof → Action structure

COLOR PALETTE (maintain but optimize usage):
- Background: #0B0F14 (Deep Charcoal)
- Primary: #00E0FF (Cyan)
- Secondary: #9EF01A (Green)
- Text: #F8FAFC (White) / #94A3B8 (Slate)

TECH STACK: React + Vite + Tailwind CSS (must use Tailwind)

AUDIT REQUIREMENTS:

1. HERO SECTION ANALYSIS
   - Does it immediately communicate value?
   - Is headline compelling and benefit-focused?
   - Is portrait/image placement optimal or should it be repositioned/removed?
   - Are CTAs clear and compelling?
   - Are trust indicators visible above the fold?
   - Provide complete recommended hero structure

2. INFORMATION ARCHITECTURE
   - Does page follow logical conversion funnel? (Awareness → Interest → Desire → Action)
   - Recommend optimal section order
   - Assess scannability and visual breaks
   - Evaluate progressive disclosure

3. VISUAL DESIGN & HIERARCHY
   - Typography scale and hierarchy
   - Whitespace and breathing room
   - Strategic color usage
   - Visual weight of important elements
   - Design consistency

4. TRUST & SOCIAL PROOF
   - Testimonial placement and credibility
   - Case study effectiveness
   - Metrics display prominence
   - Trust badges/logos needed?
   - Risk reversal elements

5. PRODUCT SHOWCASE
   - Are 4 product cards clear and compelling?
   - Is pricing clear?
   - Are features benefit-focused?
   - Need comparison table?

6. CTA STRATEGY
   - Optimal CTA placement at decision points
   - Action-oriented, benefit-focused copy
   - Button design that stands out appropriately
   - Different CTAs for different funnel stages?

7. MOBILE EXPERIENCE
   - Responsive design assessment
   - Touch target sizing
   - Mobile navigation clarity
   - Content prioritization on mobile

8. CONVERSION OPTIMIZATION
   - Friction points identification
   - Objection handling
   - Urgency/scarcity elements needed?
   - Exit-intent strategies

DELIVERABLES REQUIRED:

1. Executive Summary (1-2 pages)
   - Overall assessment
   - Top 5 critical issues
   - Quick wins (<2 hours)
   - Conversion improvement potential

2. Detailed Section-by-Section Analysis
   - Current state, what works, what doesn't
   - Specific recommendations with rationale
   - Visual mockup descriptions

3. Recommended Page Structure
   - Complete section order
   - Section purposes
   - Key content elements
   - CTA placement strategy

4. Design System Recommendations
   - Typography scale
   - Spacing system
   - Color usage guidelines
   - Component patterns
   - Animation suggestions

5. Competitive Analysis
   - Comparison to Square, Stripe, etc.
   - Elements to borrow/adapt
   - Unique differentiators to emphasize

6. Implementation Priority Matrix
   - P0: Critical must-fix issues
   - P1: High ROI improvements
   - P2: Polish and refinement
   - P3: Future experiments

7. Specific Code/Design Recommendations
   - Component structure suggestions
   - Tailwind class recommendations
   - Layout patterns
   - Copy suggestions for headlines/CTAs

SUCCESS CRITERIA:
After implementation, site should:
- Communicate value in <3 seconds
- Build trust quickly with visible social proof
- Guide clear path from awareness to action
- Reduce friction and confusion
- Feel premium (like $100k agency built it)
- Convert restaurant owners effectively

TARGET AUDIENCE INSIGHTS:
- Time-strapped, skeptical of tech
- Need quick ROI understanding
- Not tech-savvy (simplicity key)
- Respond to peer testimonials and real numbers

RESPONSE FORMAT:
1. Executive Summary
2. Critical Issues (Top 5)
3. Recommended Page Structure
4. Detailed Recommendations (by section)
5. Design System Guidelines
6. Implementation Roadmap (priority-ordered)
7. Code Examples (React/Tailwind snippets where applicable)

Be specific, actionable, and reference best practices from Square, Stripe, and other high-converting sites. Focus on conversion optimization while maintaining the "Wildly Refined" brand aesthetic.

Begin your comprehensive audit now.
```
